## 示例 不同屏幕下使用文本居中
```css
@media (max-width: 767px) {
  .text-center-xs {
  text-align: center;
  }
}
@media (min-width: 768px) and (max-width: 991px) {
  .text-center-sm {
  text-align: center;
  }
}
@media (min-width: 992px) and (max-width: 1199px) {
  .text-center-md {
  text-align: center;
  }
}
@media (min-width: 1200px) {
  .text-center-lg {
  text-align: center;
  }
}

```
